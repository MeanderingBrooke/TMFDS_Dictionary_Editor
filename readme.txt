Run Main.py to start the program
The program will run without numpy, but it is required to save and load .wav files
The dictionary editor should work without numpy


This program assumes you have finished The Message From Deep Space and have a complete dictionary
The calculator includes unit conversions for alien units, however it only uses dictionary translations for the names


On windows the dictionary save file should be in C:\Users\<user name>\AppData\LocalLow\Applesinmypants\The Message From Deep Space


If theres any issues with the program or you have any suggestions, let me know!

Good luck and have fun! :3
